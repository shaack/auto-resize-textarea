auto-resize-textarea

Auto resizes the height of any <textarea>.

  • No dependencies (don't needs jQuery)
  • Works with Bootstrap

Try and get it

  • Demo page
  • GitHub repository

Examples

HTML


<textarea class="auto-resize"></textarea>

<textarea rows="2" cols="20" class="auto-resize"></textarea>

<textarea style="padding: 10px" rows="2" cols="20" class="auto-resize"></textarea>

<textarea style="padding: 10px; width: 280px; height: 30px;" rows="2" cols="10"
                  class="auto-resize"></textarea>


<textarea id="textarea5" rows="1" cols="40" class="auto-resize"></textarea>

Script

<script src="src/auto-resize-textarea.js"></script>
<script>
    autoResizeTextarea(document.querySelectorAll("textarea.auto-resize"), {maxHeight: 120})
</script>
